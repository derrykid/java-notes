Create a treeset, after we create it, it's automatically sorted.
The code below will print ascendingly.
```java
Set<Integer> treeset = new TreeSet<>();

treeset.add(3);
treeset.add(-10);
treeset.add(3923);
treeset.add(1);
treeset.add(0);
```

* treeset.add();
* treeset.contains();
* treeset.remove();
* treeset.clear();
* treeset.size();
* treeset.isEmpty();
* treeset.first();
* treeset.last();
